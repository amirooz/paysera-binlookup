## Installation and use

```
$ git clone https://github.com/amirphp7/paysera-binlookup.git
```
```
$ cd paysera-binlookup\app
```
```
$ php app.php input.txt
```
```
$ Unit Test
```
```
$ composer install
```
**If missing any packages**
```
$ composer update
```
